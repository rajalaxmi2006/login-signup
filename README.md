# Login / Signup Page

A clean and responsive login and signup page built to provide users with a smooth and simple authentication interface. The design is optimized for desktop, tablet, and mobile devices.

## Overview

This project features a modern user authentication UI with separate login and signup forms. It is designed to be easy to use, visually appealing, and simple to customize for different web applications.

## Features

- Responsive design for all screen sizes
- Clean and modern user interface
- Login form
- Signup form
- Input fields with proper structure
- Easy to customize layout and styling
- Beginner-friendly project structure

## Tech Stack

- HTML
- CSS
